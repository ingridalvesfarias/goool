# ⚽ GOOOOL

Um jogo de pênaltis minimalista, rápido e estiloso, desenvolvido inteiramente em **HTML5, CSS3 e JavaScript Vanilla**.

## 🚀 Sobre o Projeto
O **Spider-Soccer Pro** nasceu da ideia de unir a estética de super-heróis (estilo *Spider-Man*) com uma mecânica de jogo clássica. O objetivo é testar seus reflexos contra uma IA de goleiro que se movimenta aleatoriamente no gol.

## 🎮 Como Jogar
- **Controle:** Use as setas `⬅` e `➡` na interface para posicionar a bola na mira correta.
- **Ação:** Clique no botão `CHUTAR` para disparar a bola ao gol.
- **Objetivo:** Marcar o maior número de gols possível antes de ser defendido!

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura do estádio e elementos de interface.
- **CSS3:** Estilização neon, animações (`keyframes`), design responsivo e efeitos visuais.
- **JavaScript (Vanilla):** Lógica do jogo, movimentação do goleiro (IA), detecção de colisão e manipulação do DOM.

## ⚙️ Como rodar localmente
1. Clone este repositório ou baixe o arquivo `index.html`.
2. Abra a pasta do projeto.
3. Clique duas vezes no arquivo `index.html` para abrir diretamente no seu navegador favorito.
